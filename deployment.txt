## Step 1 — MongoDB Atlas

1. Go to https://cloud.mongodb.com → create free account
2. Create a **Free M0 cluster** (any region close to you)
3. Database Access → Add User → username + password (save these)
4. Network Access → Add IP Address → **0.0.0.0/0** (allow all)
5. Clusters → Connect → Drivers → copy the connection string
   - Looks like: `mongodb+srv://user:pass@cluster0.xxxxx.mongodb.net/felicity?retryWrites=true&w=majority`

---

## Step 2 — hCaptcha (for participant registration)

The frontend already uses the **test site key** (`10000000-ffff-ffff-ffff-000000000001`).
The backend tests against the hCaptcha test secret.

For production:
1. Go to https://dashboard.hcaptcha.com → create free account
2. Add a site → copy your **Site Key**
3. Copy your **Secret Key** from Account Settings
4. Set `REACT_APP_HCAPTCHA_SITE_KEY=<your-site-key>` in Vercel
5. Set `HCAPTCHA_SECRET=<your-secret-key>` in Render

> The test keys (`10000000-ffff-...`) still work for demo/grading — skip this step if not in production.

---

## Step 3 — Backend on Render

1. Go to https://render.com → New → **Web Service**
2. Connect your GitHub repo
3. Settings:
   - **Root Directory:** `backend`
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
4. Environment Variables → Add all:

```
MONGODB_URI=mongodb+srv://user:pass@cluster0.xxxxx.mongodb.net/felicity?retryWrites=true&w=majority
JWT_SECRET=any_random_long_string_here
JWT_EXPIRE=7d
NODE_ENV=production
ADMIN_EMAIL=admin@felicity.iiit.ac.in
ADMIN_PASSWORD=YourSecureAdminPassword
HCAPTCHA_SECRET=0x0000000000000000000000000000000000000000
```

5. Click **Create Web Service** → wait for deploy (~2-3 min)
6. Copy the URL e.g. `https://felicity-api.onrender.com`

---

## Step 4 — Frontend on Vercel

1. Go to https://vercel.com → New Project
2. Import your GitHub repo
3. Settings:
   - **Root Directory:** `frontend`  
   - **Build Command:** `npm run build`
   - **Output Directory:** `build`
4. Environment Variables → Add:

```
REACT_APP_API_URL=https://felicity-api.onrender.com/api
```

5. Click **Deploy** → wait (~2 min)
6. Copy the generated URL (e.g. `https://felicity.vercel.app`)
